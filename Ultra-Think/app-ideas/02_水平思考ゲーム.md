# 水平思考ゲーム - 詳細仕様書

## 概要
ウミガメのスープ型の謎解きゲーム。AIが質問を理解し「はい」「いいえ」「関係ありません」で回答

## ターゲットユーザー
- 謎解き・推理ゲーム好き
- 論理的思考を楽しむ人
- パーティーゲームとして楽しみたい人

## コアコンセプト
**AIが出題者となる水平思考ゲーム** - いつでも一人でも楽しめる

## 主要機能

### 1. 問題出題システム
- 定番問題の収録（初期20問）
- 難易度別カテゴリー（初級・中級・上級）
- ランダム出題機能

### 2. AI質問解析エンジン
- 自然言語処理で質問を理解
- 文脈を考慮した一貫性のある回答
- 「はい」「いいえ」「関係ありません」の3択回答

### 3. ゲームモード
- **ソロプレイ**: 1人でじっくり推理
- **タイムアタック**: 制限時間内にクリア
- **対戦モード**: 質問数を競う（将来実装）

### 4. ヒント機能
- 段階的ヒント（3段階）
- ヒント使用でスコア減点
- AIによる質問サジェスト

### 5. スコアリング
- 質問数（少ないほど高得点）
- クリア時間
- ヒント使用回数
- 実績・バッジシステム

## 技術仕様

### AI実装方式
- **Phase 1**: キーワードベース（ルールベース）
- **Phase 2**: OpenAI/Claude API統合
- **Phase 3**: ファインチューニングモデル

### プラットフォーム
- 初期: Webアプリ（スマホ対応）
- 将来: iOS/Androidネイティブアプリ

### 技術スタック
- バックエンド: Python (FastAPI)
- フロントエンド: React
- AI: OpenAI API / Claude API
- データベース: SQLite → PostgreSQL

## 問題データベース設計

### 問題構造
- ID、タイトル、問題文、正解
- キーワード辞書（yes/no/irrelevant）
- 難易度、ジャンル
- ヒント文（3段階）

### 拡張性
- ユーザー投稿問題（将来実装）
- AI生成問題（実験的機能）

## 収益モデル

### 初期
- 広告モデル（1問解くごとに広告）
- 基本無料

### 将来
- 問題パック購入（100円〜）
- サブスクリプション（月額300円）
- 広告非表示オプション

## 開発フェーズ

### Phase 1: MVP（1ヶ月）
- 基本的なゲームループ
- キーワードベース回答
- 問題5問収録

### Phase 2: AI強化（2-3ヶ月）
- OpenAI/Claude API統合
- 問題20問に拡充
- スコアリング実装

### Phase 3: ゲーム性向上（4-5ヶ月）
- マルチプレイヤー
- 実績システム
- ユーザー投稿機能

## 競合分析
- 既存のウミガメのスープアプリ: 人間が出題者
- **差別化**: AI出題でいつでも遊べる

## リスクと課題
- AI回答の精度・一貫性
- APIコスト
- 問題のネタバレ対策

## 成功指標
- MAU（Monthly Active Users）
- 平均プレイ時間
- 問題クリア率
- リテンション率

## 必要リソース
- バックエンド開発者 1名
- フロントエンド開発者 1名
- AI/MLエンジニア 1名
- 問題作成者/監修者 1名
- 開発期間: 3-5ヶ月
