# ❓ よくある質問（FAQ）

## 🔰 初心者の方からよくある質問

### Q1: プログラミング経験がなくても大丈夫ですか？

**A:** はい、大丈夫です！このテンプレートは初心者向けに設計されています。
- AIアシスタント（Claude）が常にサポート
- コピー＆ペーストで動くサンプルコード
- 日本語での詳しい説明

### Q2: エラーが出て動きません

**A:** 以下の手順で解決できます：

1. **エラーメッセージをコピー**
2. **Claude（Cmd + L）に貼り付け**
3. **「初心者でもわかるように解決方法を教えて」と質問**

よくあるエラーと解決法：

| エラー | 原因 | 解決方法 |
|--------|------|----------|
| `command not found` | ツールがインストールされていない | `./setup.sh`を実行 |
| `permission denied` | 実行権限がない | `chmod +x ファイル名` |
| `module not found` | パッケージが不足 | `npm install` または `pip install -r requirements.txt` |
| `API key invalid` | APIキーが設定されていない | `.env`ファイルを確認 |

### Q3: APIキーはどこで取得できますか？

各サービスのAPIキー取得先：

| サービス | 取得先 | 料金 |
|----------|--------|------|
| OpenAI | https://platform.openai.com/api-keys | 従量課金 |
| Anthropic (Claude) | https://console.anthropic.com/ | 従量課金 |
| GitHub | Settings → Developer settings → Personal access tokens | 無料 |
| Vercel | https://vercel.com/account/tokens | 無料〜 |
| n8n | セルフホスト or クラウド版 | 無料〜 |

### Q4: どこから始めればいいですか？

**推奨の学習順序：**

1. **環境セットアップ**
   ```bash
   ./setup.sh  # これを最初に実行
   ```

2. **簡単なサンプルを動かす**
   - `examples/`フォルダのコードを実行
   - 動作を確認

3. **コードを少し変更してみる**
   - 文字を変える
   - 数値を変える
   - 色を変える

4. **AIに質問しながら機能追加**
   - 「〜する機能を追加したい」
   - 「〜を自動化したい」

### Q5: Cursorの使い方がわかりません

**基本的なショートカット：**

| 操作 | ショートカット | 説明 |
|------|---------------|------|
| AIチャット | Cmd + L | 質問や相談 |
| コード生成 | Cmd + K | 選択部分を改善 |
| 自動補完 | Tab | 提案を受け入れ |
| ファイル検索 | Cmd + P | ファイルを開く |
| 全体検索 | Cmd + Shift + F | プロジェクト内検索 |

### Q6: n8nとは何ですか？どう使いますか？

**n8nは視覚的に自動化を作れるツールです：**

1. **n8nをインストール**（無料）
   ```bash
   npm install -g n8n
   n8n start
   ```

2. **ブラウザで開く**
   - http://localhost:5678

3. **ワークフローを作成**
   - ドラッグ＆ドロップで作成
   - Webhookノードを追加
   - URLをコピー

4. **設定ファイルに記入**
   - `automation/n8n-workflows/n8n-config.json`
   - WebhookのURLを貼り付け

### Q7: Raycastとの連携方法は？

1. **Raycastをインストール**
   - https://www.raycast.com/

2. **スクリプトコマンドを追加**
   - Raycast設定 → Extensions → Script Commands
   - `scripts/raycast/`フォルダを追加

3. **使い方**
   - Raycast起動（Cmd + Space）
   - スクリプト名を入力
   - Enter で実行

### Q8: iOSアプリ開発を始めるには？

1. **Xcodeをインストール**（Mac App Store）
2. **`ios-apps/`フォルダで作業**
3. **Swiftの基礎を学ぶ**
   ```swift
   // 簡単な例
   import SwiftUI

   struct ContentView: View {
       var body: some View {
           Text("Hello, World!")
       }
   }
   ```

### Q9: データはどこに保存すればいいですか？

**用途別の推奨：**

| 用途 | サービス | 特徴 |
|------|---------|------|
| シンプルなデータ | Firebase | 設定簡単、無料枠あり |
| 本格的なDB | Supabase | PostgreSQL、無料枠あり |
| ファイル保存 | Vercel Blob | 画像や動画の保存 |
| 一時的なデータ | ローカルファイル | JSONファイルで保存 |

## 💡 Tips

### 効率的な開発のコツ

1. **まずは動くものを作る**
   - 完璧を求めない
   - 後から改善する

2. **AIを最大限活用**
   - わからないことは全て聞く
   - コードレビューも依頼

3. **小さく始める**
   - 機能を分割
   - 一つずつ実装

4. **エラーを恐れない**
   - エラーは学習のチャンス
   - AIが解決を手伝う

## 🆘 それでも解決しない場合

1. **エラーメッセージ全文をコピー**
2. **環境情報を含める**
   - OS（Mac/Windows/Linux）
   - Node.jsバージョン
   - Pythonバージョン
3. **何をしようとしたか説明**
4. **Claude/ChatGPTに相談**
