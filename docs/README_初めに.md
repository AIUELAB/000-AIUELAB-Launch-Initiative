# 🎯 Claude Code テンプレート - 初めにお読みください

## 📌 このプロジェクトについて

このテンプレートは、**開発経験がない方でも**簡単に以下のことができるように設計されています：

- 📱 iOSアプリの開発
- 🤖 業務フローの自動化
- 🌐 Webアプリケーションの作成
- 🔧 各種自動化ツールの構築

## 📁 フォルダの説明

| フォルダ名 | 用途 | 何を入れる？ |
|-----------|------|-------------|
| 📱 `ios-apps/` | iOSアプリ開発用 | Swiftファイル、Xcodeプロジェクト |
| 🤖 `automation/` | 自動化スクリプト | n8n、Raycastの設定ファイル |
| 🌐 `web-apps/` | Webアプリ用 | HTML、CSS、JavaScript、TypeScript |
| 📚 `docs/` | ドキュメント | 使い方、ガイド、メモ |
| 🧪 `examples/` | サンプルコード | 参考にできるコード例 |
| 🔧 `.configs/` | 設定ファイル | 基本的に触らなくてOK |

## 🚀 最初にやること

### 1. 環境のセットアップ（1回だけ）

```bash
# ターミナルで実行（コピー＆ペーストでOK）
./setup.sh
```

### 2. Cursorで開く

```bash
# プロジェクトをCursorで開く
code .
```

### 3. AIアシスタントを使う

- **Cmd + K**: コード生成（選択した部分を改善）
- **Cmd + L**: チャット（質問や相談）
- **Tab**: 自動補完

## 🆘 困ったときは

### エラーが出たとき

1. **エラーメッセージをコピー**
2. **Claude（Cmd + L）に貼り付けて質問**
3. **提案された解決策を試す**

### よくあるエラーと解決法

| エラー | 解決方法 |
|--------|----------|
| `command not found` | `npm install` を実行 |
| `permission denied` | `chmod +x ファイル名` を実行 |
| `module not found` | `pip install -r requirements.txt` を実行 |

## 🎯 次のステップ

1. **簡単なプロジェクトから始める**
   - `examples/`フォルダのサンプルを見る
   - コピーして改造してみる

2. **AIに聞きながら進める**
   - わからないことは全てClaude/ChatGPTに質問
   - 「初心者でもわかるように」と付け加える

3. **少しずつ理解を深める**
   - 動いたコードを読み返す
   - なぜ動くのか理解していく
